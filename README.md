# Easy-Vue
vue整合springCloud,旨在快速帮助搭建通用的web前后台框架,使用经典技术组合（Spring boot、Spring Security、MyBatis，Vue, Elementui，redis，elasticsearch,spring activity，rabbitMq），包括核心模块如：角色用户、权限控制,用户管理,文件上传,缓存管理，监控管理,日志管理等
